## 3. Cognitive Patterns Implemented
- **Judgment Loop**: The `judge-output` skill provides a reward function for self-correction.
- **Socratic Dialogue**: Relentless alignment before execution to reduce token rot.
- **Anchored Iterative Summarization**: Preserving reasoning trace across compaction.

## 4. Technical Verification
- **Safety**: Universal blocking of destructive `git` and `shell` commands.
- **Reliability**: `registry.json` (Skills.lock) ensures the harness is untampered.
- **Performance**: Shorthand communication (RTK) and AMR routing minimize costs.
